## Product name

BookmarkNest: X Bookmark Manager

## Short description

Search, organize, export, and protect X bookmarks in a local-first research library.

## Description

BookmarkNest turns X and Twitter bookmarks into a local-first research library for Microsoft Edge. Use your existing signed-in X account to import bookmarks, then search by post text, author, handle, tags, folders, and notes. Organize with folders, tags, archive, saved views, and local automation rules. Export your work to JSON, Markdown, or CSV.

Bookmark content stays in browser storage by default. Optional encrypted Cloud Sync creates a protected backup for device changes and restores. The extension encrypts backup data before upload, so plaintext bookmark content is not sent to the Cloud Sync service.

Free includes search, folders, tags, archive, delete, and JSON backup. Pro unlocks notes, saved views, Markdown and CSV export, bulk actions, background sync, mirror removals, and encrypted Cloud Sync.

BookmarkNest is currently in public testing. Test access is free and no payments are collected. Final paid plans and pricing will be set only after testing is complete.
